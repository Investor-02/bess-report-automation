# UZE Report Automation

Локальное приложение для Windows, которое будет автоматизировать подготовку отчетов УЗЕ.

## Что есть в первой версии

- Главный экран приложения.
- Модуль `РПЧ / FCR — Таблица_0`.
- Выбор станции: `Олександрійська БЕСС` или `Знаменська БЕСС`.
- Ввод среднего курса `EUR/UAH` за месяц.
- Загрузка файла `FCR_monitoring.xlsx` на уровне интерфейса.
- Кнопка `Рассчитать` без реальной логики расчета.
- Структура под будущие модули: `РДН / ВДР`, `Небалансы`, `DataHub`, `Итоговый отчет`.

## Команды

```bash
npm install
npm run dev
npm run build
npm run dev:electron
npm run dist
```

`npm run dist` подготовлен для будущей сборки Windows installer/portable через Electron Builder.
